## 一、概述

**文档信息**

| 版本号 | 日期 | 作者 |

### 1. 需求背景

**目标用户现状**：用户有多个 moomoo 账户，希望用 Python API 获取账户、持仓、期权、资产和收益数据，并在本地长期维护统一看板。

**核心痛点**：moomoo 多账户数据分散，无法一眼看到总仓位、主题暴露、期权风险、收益表现和历史变化。

| 问题点 | 问题描述 | 当前影响 |
|---|---|---|
| 多账户分散 | 股票、期权、现金分布在多个账户 | 难以判断整体仓位和风险 |
| 主题归因缺失 | 投资主题如太空、AI、医疗需要人工理解 | 难以看清配置是否过度集中 |
| 历史不可回看 | 当前还没有 SQLite 快照体系 | 无法查看收益曲线和仓位变化 |
| 期权风险不直观 | 卖出看跌等仓位需要单独识别 | 难以及时判断潜在资金占用和风险 |

### 2. 需求目标

**业务目标**

- 在本地 Web App 中合并展示多个 moomoo 账户的当前资产、持仓、收益和期权风险。
- 每次同步生成 SQLite 快照，支持按历史时间查看看板。
- 基于历史快照生成总资产曲线、累计收益曲线和日收益曲线。

**成功标准**

| 指标 | 当前状态 | 目标状态 |
|---|---|---|
| 多账户汇总 | 未实现 | 可在一个页面查看总览 |
| 历史快照 | 未实现 | 每次同步后写入 SQLite |
| 收益曲线 | 未实现 | 可按历史快照自动生成 |
| 主题板块 | 未实现 | 可通过本地配置聚合展示 |
| 期权风险 | 分散查看 | 可独立列出卖出看跌等风险仓位 |

### 3. 需求范围

**包含**

- 多 moomoo 账户数据同步
- SQLite 历史快照
- 当前/历史快照切换
- 总资产、收益、现金、股票、期权占比
- 个股仓位树图

- AI 诊断
- 自动智能主题识别
- 交易下单功能

## 三、页面结构与导航设计

### 整体导航结构

```text
单页面看板
├─ 顶部控制区 -> 同步数据、选择历史快照、查看更新时间
├─ 资产摘要区 -> 总资产、总收益、日收益、股票/期权/现金占比
├─ 持仓地图区 -> 个股仓位树图
├─ 板块明细区 -> 人工主题聚合后的主题占比和明细
├─ 收益曲线区 -> 历史总资产、累计收益、日收益曲线

```

### 单页面看板

**页面职责**：让用户在一个页面完成当前资产判断、历史回看、主题配置判断和期权风险检查。

| 区域 | 内容 | 显示条件 |
|---|---|---|
| 顶部控制区 | 同步按钮、快照时间选择器、当前数据时间 | 始终显示 |
| 资产摘要区 | 总资产、总收益、日收益、股票占比、期权占比、现金占比 | 有快照时显示 |
| 持仓地图区 | 按市值占比展示个股树图 | 有股票或 ETF 持仓时显示 |
| 板块明细区 | 主题卡片、主题占比、主题内标的列表 | 有持仓时显示 |
| 收益曲线区 | 总资产曲线、累计收益曲线、日收益柱状/折线 | 至少 2 个历史快照时显示 |
| 期权风险区 | 标的、类型、方向、行权价、到期日、市值/义务 | 有期权持仓时显示 |
| 空状态 | 引导同步数据 | 无快照时显示 |

**本页面不展示**：交易下单、账户登录、会员分享、AI 建议、云端同步设置。

## 四、功能说明

### 功能 1：多账户同步与快照写入

**功能描述**

用户可在单页面看板点击同步按钮，从 moomoo OpenD / Python API 拉取多个账户的资产、持仓、期权和收益数据，合并后写入 SQLite 快照。

**用户流程**

```text
点击同步
  ↓
连接 moomoo OpenD
  ↓
逐个账户拉取资产、持仓、期权、收益
  ├─ 成功 -> 合并数据并写入 SQLite -> 刷新看板
  ├─ 失败（OpenD 未启动）-> 显示连接失败
  ├─ 失败（账户无权限）-> 跳过该账户并标记异常
  └─ 失败（写入数据库失败）-> 保留当前页面并显示错误
```

**状态机**

| 状态名称 | 进入条件 | 退出条件 | 用户可见表现 |
|---|---|---|---|
| 未同步 | 本地无快照 | 同步成功 | 显示空状态 |
| 同步中 | 点击同步 | 成功或失败 | 显示进度弹窗 |
| 同步成功 | API 与数据库写入成功 | 下一次同步 | 更新快照时间 |
| 部分成功 | 部分账户失败 | 用户关闭提示 | 显示异常账户数量 |
| 同步失败 | 全部账户失败或数据库失败 | 重试同步 | 显示失败原因 |

**字段规范**

| 字段名 | 类型 | 是否必填 | 长度限制 | 校验规则 | 错误提示 |
|---|---|---|---|---|---|
| account_id | 文本 | 是 | 64 | 不为空 | 账户 ID 为空 |
| snapshot_time | 日期时间 | 是 | ISO 时间 | 不晚于当前时间 | 快照时间异常 |
| symbol | 文本 | 是 | 32 | 不为空 | 标的代码为空 |
| quantity | 数字 | 是 | 无 | 可为正负 | 持仓数量异常 |
| market_value | 数字 | 是 | 无 | 可为正负 | 市值数据异常 |
| pnl | 数字 | 否 | 无 | 可为正负 | 收益数据异常 |

**文案规范**

| 场景 | 文案内容 |
|---|---|
| 空状态 | 暂无快照，请先同步 moomoo 账户数据 |
| 加载中 | 正在同步账户数据 |
| 操作成功 | 同步完成，已生成新快照 |
| 操作失败 | 同步失败，请检查 moomoo OpenD 状态 |
| 权限不足 | 部分账户无法读取，请检查账户权限 |
| 网络异常 | 连接 moomoo OpenD 失败，请确认服务已启动 |

**异常情况处理**

| 异常场景 | 触发条件 | 处理方式 | 用户反馈 |
|---|---|---|---|
| 网络断开 | OpenD 请求失败 | 停止同步并保留旧快照 | 连接失败，请稍后重试 |
| 数据为空 | API 返回空持仓 | 写入空持仓快照 | 当前账户暂无持仓 |
| 操作超时 | 单账户超过 15 秒 | 跳过该账户 | 账户同步超时 |
| 权限不足 | API 返回权限错误 | 标记账户失败 | 账户权限不足 |

### 功能 2：当前/历史持仓全景

**功能描述**

用户可在单页面看板选择当前或历史快照，查看对应时间点的总资产、收益、仓位占比、个股树图和主题板块。

**用户流程**

```text
打开看板
  ↓
读取最新快照
  ↓
展示资产摘要和持仓地图
  ├─ 选择历史快照 -> 重新渲染该时间点数据
  ├─ 无快照 -> 显示空状态
  └─ 快照损坏 -> 显示读取失败
```

**状态机**

| 状态名称 | 进入条件 | 退出条件 | 用户可见表现 |
|---|---|---|---|
| 加载快照 | 页面打开 | 读取成功或失败 | 骨架屏 |
| 最新快照 | 默认读取成功 | 选择历史快照 | 显示最新数据 |
| 历史快照 | 选择某个时间点 | 选择其他快照 | 显示历史标记 |
| 无数据 | SQLite 无快照 | 同步成功 | 空状态 |
| 读取失败 | 数据库读取异常 | 重试成功 | 错误提示 |

**字段规范**

| 字段名 | 类型 | 是否必填 | 长度限制 | 校验规则 | 错误提示 |
|---|---|---|---|---|---|
| snapshot_id | 整数 | 是 | 无 | 必须存在于数据库 | 快照不存在 |
| total_assets | 数字 | 是 | 无 | 大于等于 0 | 总资产异常 |
| stock_ratio | 百分比 | 是 | 0-999% | 可超过 100% | 股票占比异常 |
| option_ratio | 百分比 | 是 | -999%-999% | 允许负值 | 期权占比异常 |
| cash_ratio | 百分比 | 是 | -999%-999% | 允许负值 | 现金占比异常 |

**文案规范**

| 场景 | 文案内容 |
|---|---|
| 空状态 | 暂无历史快照 |
| 加载中 | 正在读取快照 |
| 操作成功 | 已切换到所选快照 |
| 操作失败 | 快照读取失败 |
| 权限不足 | 本地数据不可访问 |
| 网络异常 | 当前页面读取本地数据，无需网络 |

**异常情况处理**

| 异常场景 | 触发条件 | 处理方式 | 用户反馈 |
|---|---|---|---|
| 数据为空 | 无快照 | 显示同步入口 | 请先同步数据 |
| 快照不存在 | 选择已删除快照 | 回退最新快照 | 该快照不存在 |
| 数据损坏 | JSON/记录解析失败 | 跳过异常记录 | 部分数据无法显示 |
| 操作超时 | 查询超过 5 秒 | 中止查询 | 读取超时，请重试 |

### 功能 3：收益曲线

**功能描述**

用户可在单页面看板查看基于 SQLite 历史快照生成的总资产曲线、累计收益曲线和日收益曲线。

**用户流程**

```text
进入收益曲线区
  ↓
读取历史快照
  ↓
按时间排序生成曲线
  ├─ 快照数量足够 -> 展示曲线
  ├─ 快照少于 2 条 -> 显示数据不足
  └─ 收益字段缺失 -> 使用资产变化兜底计算并标记
```

**状态机**

| 状态名称 | 进入条件 | 退出条件 | 用户可见表现 |
|---|---|---|---|
| 数据不足 | 快照少于 2 条 | 新增快照 | 显示提示 |
| 曲线可用 | 快照至少 2 条 | 数据异常 | 显示图表 |
| 兜底计算 | 收益字段缺失 | 新收益数据同步 | 显示计算标记 |
| 曲线失败 | 历史数据读取失败 | 重试成功 | 显示错误 |

**字段规范**

| 字段名 | 类型 | 是否必填 | 长度限制 | 校验规则 | 错误提示 |
|---|---|---|---|---|---|
| date | 日期时间 | 是 | ISO 时间 | 按时间递增 | 时间序列异常 |
| total_assets | 数字 | 是 | 无 | 大于等于 0 | 总资产缺失 |
| total_pnl | 数字 | 否 | 无 | 可为正负 | 收益数据缺失 |
| daily_pnl | 数字 | 否 | 无 | 可为正负 | 日收益数据缺失 |

**文案规范**

| 场景 | 文案内容 |
|---|---|
| 空状态 | 历史快照不足，至少需要两次同步 |
| 加载中 | 正在生成收益曲线 |
| 操作成功 | 收益曲线已更新 |
| 操作失败 | 收益曲线生成失败 |
| 权限不足 | 无法读取本地历史数据 |
| 网络异常 | 当前图表读取本地数据，无需网络 |

**异常情况处理**

| 异常场景 | 触发条件 | 处理方式 | 用户反馈 |
|---|---|---|---|
| 数据为空 | 无历史快照 | 显示空状态 | 请先同步多次数据 |
| 字段缺失 | pnl 缺失 | 用资产变化计算 | 已使用资产变化估算 |
| 操作超时 | 查询超过 5 秒 | 限制时间范围 | 历史数据较多，请缩小范围 |
| 数据异常 | 资产为负或时间重复 | 跳过异常点 | 部分异常快照已跳过 |

### 功能 4：期权持仓与风险暴露

**功能描述**

用户可在单页面看板查看期权持仓列表，重点识别卖出看跌期权、到期日、行权价、名义义务和关联主题。

**用户流程**

```text
进入期权风险区
  ↓
读取快照中的期权持仓
  ↓
解析方向、类型、标的、到期日、行权价
  ├─ 有卖出看跌 -> 高亮风险暴露
  ├─ 无期权 -> 显示无期权持仓
  └─ 合约解析失败 -> 放入异常合约列表
```

**状态机**

| 状态名称 | 进入条件 | 退出条件 | 用户可见表现 |
|---|---|---|---|
| 无期权 | 快照无期权 | 新快照有期权 | 显示空状态 |
| 正常展示 | 期权解析成功 | 切换快照 | 显示列表 |
| 风险高亮 | 存在卖出看跌 | 仓位消失 | 高亮标签 |
| 解析异常 | 合约字段不完整 | 数据修复 | 显示异常列表 |

**字段规范**

| 字段名 | 类型 | 是否必填 | 长度限制 | 校验规则 | 错误提示 |

| option_type | 枚举 | 是 | CALL/PUT | 只能为 CALL 或 PUT | 期权类型异常 |
| side | 枚举 | 是 | LONG/SHORT | 只能为 LONG 或 SHORT | 期权方向异常 |
| strike | 数字 | 是 | 无 | 大于 0 | 行权价异常 |
| expiry | 日期 | 是 | YYYY-MM-DD | 不早于快照日期 | 到期日异常 |
| notional_exposure | 数字 | 是 | 无 | 大于等于 0 | 风险暴露异常 |

**文案规范**

| 场景 | 文案内容 |
|---|---|
| 空状态 | 当前快照无期权持仓 |
| 加载中 | 正在读取期权持仓 |

| 操作失败 | 期权数据解析失败 |

### 功能 5：本地主题映射

**功能描述**

用户可在主题配置弹窗维护股票代码与主题的对应关系；主题板块按该配置聚合，未配置标的归入“未分类”，期权跟随正股主题。

**用户流程**

```text
打开主题配置
  ↓
新增或编辑 symbol -> theme
  ↓
保存配置
  ├─ 成功 -> 重新计算主题板块
  ├─ symbol 重复 -> 提示覆盖或修改
  └─ theme 为空 -> 阻止保存
```

**状态机**

| 状态名称 | 进入条件 | 退出条件 | 用户可见表现 |
|---|---|---|---|
| 查看配置 | 打开弹窗 | 编辑条目 | 显示映射表 |
| 编辑中 | 修改字段 | 保存或取消 | 显示未保存状态 |
| 保存成功 | 校验通过 | 关闭弹窗 | 看板重新聚合 |
| 保存失败 | 数据库写入失败 | 重试或取消 | 错误提示 |

**字段规范**

| 字段名 | 类型 | 是否必填 | 长度限制 | 校验规则 | 错误提示 |
|---|---|---|---|---|---|
| symbol | 文本 | 是 | 32 | 大写字母/数字/点/横线 | 标的代码格式不正确 |
| theme | 文本 | 是 | 30 | 不为空 | 主题名称不能为空 |
| display_name | 文本 | 否 | 50 | 可为空 | 显示名称过长 |
| color | 文本 | 否 | 7 | HEX 色值 | 颜色格式不正确 |

**文案规范**

| 场景 | 文案内容 |
|---|---|
| 空状态 | 暂无主题配置，未配置标的将进入未分类 |
| 加载中 | 正在读取主题配置 |
| 操作成功 | 主题配置已保存 |
| 操作失败 | 主题配置保存失败 |
| 权限不足 | 无法写入本地配置 |
| 网络异常 | 主题配置为本地数据，无需网络 |

**异常情况处理**

| 异常场景 | 触发条件 | 处理方式 | 用户反馈 |
|---|---|---|---|
| symbol 重复 | 新增已有 symbol | 阻止保存 | 该标的已存在 |
| 主题为空 | theme 为空 | 阻止保存 | 请输入主题名称 |
| 数据库失败 | 写入 SQLite 失败 | 保留编辑内容 | 保存失败，请重试 |
| 未配置标的 | 持仓无映射 | 归入未分类 | 部分标的未分类 |
